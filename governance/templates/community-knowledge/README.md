# Community Knowledge

This folder contains local context that the community AI uses to be helpful. Community members contribute knowledge through the bot or by editing these files directly.

## Files

- **events.md** — Current and upcoming events
- **spaces.md** — Physical and virtual spaces
- **resources.md** — Shared resources, links, tools
- **decisions.md** — Recent governance decisions
- **norms.md** — Informal community norms

## How it works

The AI reads these files for context when answering questions. When community members share useful info ("the sauna is broken", "dinner is at 7pm"), the AI can offer to add it here.

What gets stored is governed by the constitution. The AI will not store personal health info, private conversations, or anything the community rules prohibit.
